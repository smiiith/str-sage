# STR Sage
STR Sage is a data-driven analysis platform for short-term rental (STR) hosts and investors. It offers three core tools:

  - Feedback Genius – AI-powered analysis of existing Airbnb listings, evaluating title, description, amenities, photos, and interior design to help hosts improve performance.
  - Market Spy – Competitive market research for potential STR investment properties, analyzing comparable listings and occupancy data to assess viability.
  - Market Scout – Streamlined market analysis focused on entire home STR properties.
  - Guides -- Helpful guides provide help on topics like Airbnb host fees, photos and amenities.

  Built for Airbnb hosts and real-estate investors who want to make smarter, data-backed decisions about short-term rentals.
